# 对齐方式

---

<common-code-format>
  <docsComponents-TTable-align slot="source"></docsComponents-TTable-align>
配置

  在`TTable`组件中添加`align:'center'`设置整个 table 对齐方式

  在`columns`中设置`align`设置每列对齐方式（`columns优先级`高于`整体的对齐方式`）----`align`可选值`left`、`center`、`right`<br/>
